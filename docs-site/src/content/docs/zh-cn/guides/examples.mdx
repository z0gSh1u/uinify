仓库不再保留推测性质的 app templates。当前 example surface 有意收窄为一个可运行 Vite examples app 和一个保留的 OpenAI-like mapper。

## 当前文件

- `examples/src/App.tsx`：本地 examples app shell 和路由注册
- `examples/playground/App.tsx`：`/playground` 路由，把 OpenAI-like chunks 映射成 `UiStreamEvent`，再渲染 runtime transcript
- `examples/adapters/openai-like.ts`：OpenAI-like stream shape 的 reference-only mapper

本地运行 examples app：

```bash
pnpm dev:example
```

## 如何阅读示例

把 `/playground` 当作 public runtime 和 transcript renderer 的 smoke check。把 mapper 当作协议规范化应该放在什么位置的参考。

不要把这个 mapper 当成受支持的 OpenAI 兼容层。真实产品应该在宿主边界把自己的后端 payloads 映射成 `UiStreamEvent`。
