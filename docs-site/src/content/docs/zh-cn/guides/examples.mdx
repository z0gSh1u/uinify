示例模板是可复制的起点，不是新的 API。用它们选择集成形态，然后把生产代码锚定在公开 package surface 上。

## 选择起点

- `examples/templates/minimal-app.tsx`：最小 runtime 与 transcript wiring
- `examples/templates/adapter-template.tsx`：使用 `createAdapterRunner` 映射宿主事件
- `examples/templates/upload-template.tsx`：受控附件生命周期
- `examples/templates/artifact-template.tsx`：artifact renderer overrides

## 常见路径

如果你只是验证渲染，请从 minimal template 开始：

```tsx
import { useState } from "react"
import { createChatRuntime } from "uinify"
import { ChatRoot, MessageList } from "uinify/react"

export function App() {
  const [runtime] = useState(() => createChatRuntime({ conversationId: "demo" }))

  return (
    <ChatRoot runtime={runtime}>
      <MessageList />
    </ChatRoot>
  )
}
```

如果你已经有后端协议，请切到 adapter template，在 dispatch 前先规范化宿主事件。

## 如何阅读示例

- 把 `examples/templates/*` 当成由 docs 支撑的 recipes。
- 把 `examples/adapters/*` 当成仅供参考的 protocol mappers。
- 复制模式，而不是复制 fixture id 或示例标签。
