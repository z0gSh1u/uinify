Welcome to uinify — a React library for chat UIs that separates transport from rendering.

Map your backend events into `UiStreamEvent`, dispatch them into a runtime, and render the conversation.

## Quick Start

```bash
npm install uinify react react-dom
```

Read the [Getting Started](/getting-started/) guide to set up your first chat UI.
