# Getting Started

## Goal

Render a runtime-backed transcript inside an existing React app.

## Minimal Setup

```tsx
import { createChatRuntime } from "uinify"
import { ChatRoot, MessageList } from "uinify/react"

const runtime = createChatRuntime({ conversationId: "demo" })

export function DemoChat() {
  return (
    <ChatRoot runtime={runtime}>
      <MessageList />
    </ChatRoot>
  )
}
```

## What Is Stable

- `UiStreamEvent`
- `createChatRuntime`
- `ChatRoot`
- `MessageList`
```

Create `docs/integration/stream-mapping.md`

```md
# Stream Mapping

## Goal

Map backend events into `UiStreamEvent` before they reach the runtime.

## Pattern

```ts
for (const hostEvent of hostEvents) {
  for (const event of mapOpenAiLikeChunk(hostEvent)) {
    runtime.dispatch(event)
  }
}
```

## Rule

Mapping ends at `UiStreamEvent`. Do not pass vendor-specific payloads into `uinify` state.
```

# Upload Lifecycle

## Host-owned Uploads

`uinify` renders upload state. Your host app performs the upload.

## Composer Pattern

```tsx
<LexicalComposer
  attachments={attachments}
  onAttachmentsChange={setAttachments}
  onAttachmentRetry={retryAttachment}
  sendPolicy="uploaded-only"
  onSubmit={handleSubmit}
/>
```

## Required States

- `queued`
- `uploading`
- `uploaded`
- `error`
- `removed`
```
